# Iris Flower Classification

## Project Overview
This project uses Machine Learning to classify Iris flowers into three species:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

## Objective
To build a classification model that predicts the species of an Iris flower based on its measurements.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Matplotlib

## Dataset
The Iris dataset contains:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Machine Learning Algorithm
Random Forest Classifier

## Steps Performed
1. Loaded the Iris dataset.
2. Split data into training and testing sets.
3. Trained a Random Forest model.
4. Evaluated model accuracy.
5. Generated classification report and confusion matrix.

## How to Run

```bash
pip install -r requirements.txt
python iris_classification.py
```

## Output
- Classification Accuracy
- Classification Report
- Confusion Matrix
- Species Prediction
